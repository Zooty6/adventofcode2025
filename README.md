# [Advent of Code 2025](https://adventofcode.com/2025/)


The project requires java 25 to be installed!


Running the project with `./gradlew run` executes all implemented day's solutions and prints it to the console.  
To run the project, `./gradlew build` needs to download all your input files, 
which needs your AoC session key set up in `AOC_SESSION` environment variable!
